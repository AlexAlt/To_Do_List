To Do List
  Application allows you to create to do lists and to add tasks to them.

  by Jessica Joly and Alex Alteri

  Description:

  This application allows users to create to-do lists that stores tasks and their descriptions.

Technology Used:

HTML JavaScript JQuery  CSS
